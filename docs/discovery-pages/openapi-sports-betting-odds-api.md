# OpenAPI Sports Betting Odds API

`odds-api/odds-api` provides a public OpenAPI contract for sports betting odds workflows.

Canonical GitHub repo: https://github.com/odds-api/odds-api

## Public contract

- `openapi.yaml` is the canonical API contract
- `openapi.json` is the JSON export
- production base URL is `https://api.odds-api.net/v1`
- server-side auth uses `X-API-Key`

Use the OpenAPI spec to:

- generate clients
- validate requests
- inspect schemas
- understand endpoint parameters and response shapes
- ground AI coding agents in the API contract

The repo also includes plain REST examples, TypeScript SDK, Python SDK, MCP server, Postman collection, mock mode, and examples for arbitrage, positive EV, bookmaker comparison, and line movement.
