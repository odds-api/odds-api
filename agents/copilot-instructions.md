# Copilot Instructions

Build against `openapi.yaml` and the SDKs in this repo. Use API-key auth first. Design the requested app or workflow rather than copying bundled examples; examples are optional references. Prefer snapshots for one-off answers; use `odds_api.get_streaming_info` before realtime code, and use persistent MCP stream tools only for inspection workflows. Add responsible gambling and execution-risk language to user-facing arbitrage or positive EV features.
